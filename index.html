Retrospect-GDS

Retrospect-GDS is an easy to use software application for displaying your genealogy information on the web. Retrospect-GDS is designed to be flexible so that new features can be added easily. A templating system is included to make it easy to modify the look and feel. Many popular genealogy reports are available. Retrospect-GDS also supports multiple languages.

Retrospect-GDS is written entirely in PHP and supports multiple databases, including MySQL, PostgreSQL 7.x, and Oracle.
